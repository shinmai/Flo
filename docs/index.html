Getting started

Adding Flo to Slack workplace

First, you should add Flo to your workplace. To do that, click here and authorize app.

Flo should appear in Apps tab. You may want to add him to different channels.

Configuring notifications

To receive Glo notifications, you need to obtain Webhook URL and Secret from Flo's settings. These will vary depending on a channel you are currently in. Be sure to be in the right one!

Type /flo, go to Settings and copy these values.

Now open your board's settings, head to Webhooks and add new. Type appropriate name, both URL and Secret which you saved earlier and select which notifications you'd like to receive. Notifications should start flowing in!

Changing language

English doesn't suit you? To change language go to Settings tab like you did in previous step and select one from dropdown menu.

Currently supported languages are English and Polish.

We don't store your personal informations

We have nothing to store, except your channel preferences (preferred language and secret used tu verify incoming webhooks) and workplace's auth token (used to send notifications). We aren't, however, responsible for any data stored by third party apps like Glo and Slack..
